HTML Practice

Exercise 1: Create Your Profile

Build a simple profile with:

  • Your name as an H1
  • A short bio paragraph
  • A list of 3 hobbies

Exercise 2: Favorite Things

Create two lists:

  • Unordered list of 3 favorite movies
  • Ordered list of top 3 Pokemon

Exercise 3: Add Links and Images

Create:

  • A link to your favorite website
  • An image from the internet (use placeholder: https://via.placeholder.com/200)
  • Make the image also a link (wrap img in anchor tag)

Exercise 4: Create a Table

Make a table with 3 columns (Name, Age, City) and 3 rows of data

Exercise 5: Build a Contact Form

Create a form with:

  • Name input (text)
  • Email input (email type)
  • Message textarea
  • Submit button

Exercise 6: Use Semantic HTML

Create a page structure with:

  • A header with your name
  • A nav with 3 dummy links (Home, About, Contact)
  • A main section with an article containing a paragraph
  • A footer with copyright text

Challenge 1: Build a Recipe Page

Create a complete recipe page with:

  • Recipe name as H1
  • An image of the dish
  • Unordered list of ingredients
  • Ordered list of cooking steps
  • Preparation time and servings info

Challenge 2: Pokemon Card

Build a Pokemon card using only HTML:

  • Pokemon name as H2
  • Image of Pokemon
  • Table with stats (HP, Attack, Defense)
  • List of abilities
  • Paragraph describing the Pokemon

Challenge 3: Create a Mini Portfolio

Build a simple portfolio page with:

  • Header with your name and tagline
  • Nav bar with: About, Projects, Contact
  • Main section with:
    • About section (paragraph about yourself)
    • Projects section (list of 2-3 projects with descriptions)
    • Contact section (contact form)
  • Footer with links to GitHub

Tips for Practicing:

  1. Save this file after making changes
  2. Refresh your browser to see updates
  3. Use "Inspect Element" (F12) to see your HTML structure
  4. Try breaking things on purpose to learn
  5. Use W3Schools as reference when stuck
  6. Don't worry about it looking ugly - CSS comes next!
