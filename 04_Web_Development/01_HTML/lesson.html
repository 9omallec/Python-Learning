Semantic HTML (Important!)

Modern HTML uses semantic tags that describe their content:

This is a HEADER element

Used for introductory content or navigation

NAV = Navigation links

Home | About | Contact

ARTICLE element

Independent, self-contained content (blog post, news article, etc.)

SECTION element

Thematic grouping of content

ASIDE = Sidebar content (related but not main)

FOOTER = Bottom content (copyright, links, etc.)

Key Takeaways

  • HTML provides the STRUCTURE of a webpage
  • Tags are wrapped in angle brackets: <tagname>
  • Most tags need opening and closing: <p>text</p>
  • Attributes add extra info: <a href="url">
  • Semantic tags make your code meaningful and accessible
  • HTML alone is just structure - CSS makes it pretty, JS makes it interactive
