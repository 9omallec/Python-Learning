JavaScript Practice Exercises

Instructions:

  • Complete each exercise by writing JavaScript code
  • Open browser console (F12) to see console.log() output
  • Test your code by clicking the "Test" buttons
  • Check hints if you're stuck

Exercise 1: Variables and Basic Math

Task: Create variables for your name, age, and favorite number. Calculate your age in 10 years.

Instructions:

  1. Create variables: name, age, favoriteNumber
  2. Calculate ageIn10Years
  3. Use console.log() to print: "My name is [name], I am [age] years old, and in 10 years I'll be [ageIn10Years]"
Hint: Use let for variables. Use template literals: `My name is ${name}`

Exercise 2: Functions

Task: Create a function that greets a user and tells them if they're an adult.

Instructions:

  1. Write a function greetUser(name, age)
  2. It should return: "Hello [name]! You are an adult." if age >= 18
  3. Otherwise: "Hello [name]! You are not an adult yet."
Hint: Use if/else inside the function. Remember to return the string!

Exercise 3: Arrays

Task: Work with an array of Pokemon names.

Instructions:

  1. Create an array: pokemon = ["Pikachu", "Charizard", "Bulbasaur"]
  2. Add "Squirtle" to the array using .push()
  3. Use .map() to create a new array where each name is uppercase
  4. Use .filter() to find Pokemon with names longer than 8 characters
  5. Log all results to console
Hint: name.toUpperCase() makes strings uppercase. name.length gets string length.

Exercise 4: Objects

Task: Create a Pokemon object with properties.

Instructions:

  1. Create an object with: name, type, level, hp, moves (array)
  2. Add a method attack() that logs: "[name] uses [first move]!"
  3. Add a method levelUp() that increases level by 1
  4. Test both methods
Hint: Methods are functions inside objects. Use this.propertyName to access properties.

Exercise 5: Loops

Task: FizzBuzz in JavaScript! (Remember this?)

Instructions:

  1. Loop from 1 to 20
  2. If divisible by 3: log "Fizz"
  3. If divisible by 5: log "Buzz"
  4. If divisible by both: log "FizzBuzz"
  5. Otherwise: log the number
Hint: Use a for loop. Check divisible by 15 first, then 3, then 5.

Exercise 6: DOM Manipulation - Click Counter

Task: Make a button that counts clicks.

Clicks: 0

Instructions:

  1. Add an event listener to #clickButton
  2. Each click should increase a counter
  3. Update the #clickCount span with the new count
  4. Implement the reset function
Hint: Use document.getElementById() and .addEventListener('click', function)

Exercise 7: Input Validation

Task: Validate user input in real-time.

Instructions:

  1. Listen for 'input' events on the text field
  2. If username is 4-12 characters: show green "Valid!" message
  3. If too short: show red "Too short!" message
  4. If too long: show red "Too long!" message
Hint: Use .addEventListener('input', ...) and value.length

Exercise 8: Background Color Changer

Task: Change the background color of this exercise box.

Instructions:

  1. Write the changeColor(color) function
  2. It should change the background color of this exercise div
  3. Hint: This div has a class "exercise"
Hint: Use document.querySelectorAll('.exercise')[7] to get this div, then change .style.backgroundColor

Challenge 1: Temperature Converter

Task: Build a Celsius to Fahrenheit converter.

Instructions:

  1. Get the value from the input field
  2. Convert to Fahrenheit: F = (C × 9/5) + 32
  3. Display result: "[C]°C = [F]°F"
  4. Handle empty input with an error message

Task: Build a to-do list app.

    Instructions:
