## Project 1: Favorite Things List (10 min)

**Goal:** Store your favorite things and display them

**Requirements:**
- Create 3 lists: favorite_foods, favorite_games, favorite_movies
- Put 3-5 items in each list
- Print each list with a nice header
- Print the first and last item from each list
- Print how many items in each list

**Example Output:**
```
=== MY FAVORITE THINGS ===

Foods:
- pizza
- tacos
- sushi
- burgers
Total foods: 4
First: pizza, Last: burgers

Games:
- Zelda
- Mario Kart
- Minecraft
Total games: 3
First: Zelda, Last: Minecraft
```

**Bonus challenges:**
- Ask user for their favorites using input()
- Check if a specific item is in the list
- Print items in reverse order

## Project 2: Score Tracker (12 min)

**Goal:** Track game scores and find statistics

**Requirements:**
- Create a list of 7-10 game scores (numbers)
- Find and print the highest score
- Find and print the lowest score
- Calculate and print the average score
- Count how many scores are above 80

**Example Output:**
```
Game Scores: [75, 92, 88, 95, 73, 85, 90]

Highest Score: 95
Lowest Score: 73
Average Score: 85.4
Scores above 80: 4
```

**Bonus challenges:**
- Find the difference between highest and lowest
- Determine if the average is above 80
- Print each score with "Pass" or "Fail" (passing is 75+)

## Project 3: Shopping List Checker (12 min)

**Goal:** Check if items are on your shopping list

**Requirements:**
- Create a shopping list with 5-7 items
- Ask user for an item to check
- Tell them if it's on the list or not
- Show them the full list
- Tell them what number (index + 1) the item is if found

**Example Output:**
```
Shopping List:
1. milk
2. eggs
3. bread
4. cheese
5. apples

What item are you looking for? bread

‚úì Yes! Bread is on the list (item #3)

--- OR ---

```

**Bonus challenges:**
- Let them check multiple items
- Show items before and after the found item
- Count total items on list

## Project 4: Class Roster (12 min)

**Goal:** Store student names and access them by number

**Requirements:**
- Create a list of 8-10 student names
- Print the full roster with numbers (1, 2, 3...)
- Ask user for a number
- Show that student's name
- Show first 3 students
- Show last 3 students

**Example Output:**
```
=== CLASS ROSTER ===
1. Alice
2. Bob
3. Charlie
4. Diana
5. Ethan

Enter student number: 3
Student #3: Charlie

First 3 students: Alice, Bob, Charlie
Last 3 students: Charlie, Diana, Ethan
```

**Bonus challenges:**
- Ask for a name and show their number
- Show every other student (alternating)
- Count total students

## Project 5: Number Analyzer (15 min)

**Goal:** Analyze a list of numbers

**Requirements:**
- Create a list of 10-15 random numbers (use a mix of values)
- Calculate and display:
  - Total sum
  - Average
  - Highest number
  - Lowest number
  - Range (highest - lowest)
  - Count of even numbers
  - Count of odd numbers

**Example Output:**
```
Numbers: [12, 7, 23, 45, 8, 34, 19, 2, 56, 31]

=== ANALYSIS ===
Total Sum: 237
Average: 23.7
Highest: 56
Lowest: 2
Range: 54
Even numbers: 5
Odd numbers: 5
```

**Bonus challenges:**
- Count numbers above average
- Find median (middle value)
- Show numbers in groups of 5
