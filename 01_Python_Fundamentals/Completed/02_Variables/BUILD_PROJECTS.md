## Project 2: Simple Calculator (15 min)

**Goal:** Calculate something using variables

**Requirements:**
- Create at least 3 number variables
- Perform 4+ operations (+, -, *, /)
- Store results in variables
- Print results with labels

**Ideas:**
- Calculate area and perimeter of rectangle
- Calculate total cost with tax and tip
- Calculate age in days, hours, minutes
- Convert measurements (miles to km, etc.)

**Example:**
```python
# Rectangle calculator
length = 10
width = 5
area = length * width
perimeter = 2 * (length + width)

print(f"Area: {area}")
print(f"Perimeter: {perimeter}")
```

## Project 6: Age Calculator (20 min)

**Goal:** Calculate various age-related facts

**Requirements:**
- Birth year variable
- Current year variable
- Calculate current age
- Calculate age in different units (months, days, hours)
- Calculate age in 10 years, 20 years

**Example:**
```python
birth_year = 1990
current_year = 2025
age = current_year - birth_year

age_in_months = age * 12
age_in_days = age * 365
age_in_hours = age_in_days * 24

print(f"You are {age} years old")
print(f"That's {age_in_months} months")
print(f"Or {age_in_days} days!")
```
