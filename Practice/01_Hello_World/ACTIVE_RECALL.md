## Card 1: Basic Print

**Q: Write code that prints "Hello World"**

<details>
<summary>Click to reveal answer</summary>

```python
print("Hello World")
```

**Key points:**
- Use `print()` function
- Text goes in quotes (single or double)
- Quotes are required for text
</details>

## Card 2: Print Numbers

**Q: What's the difference between these two?**

```python
print("42")
print(42)
```

<details>
<summary>Click to reveal answer</summary>

**Both print `42` but:**
- `print("42")` - String (text)
- `print(42)` - Number (integer)

**The quotes make it text, not a number!**
</details>

## Card 3: Multiple Prints

**Q: What will this output?**

```python
print("Line 1")
print("Line 2")
print("Line 3")
```

<details>
<summary>Click to reveal answer</summary>

**Output:**
```
Line 1
Line 2
Line 3
```

**Each print() creates a new line automatically**
</details>

---

## Card 4: Print Multiple Items

**Q: Write code that prints: `Name: Connor Age: 35` in one line**

<details>
<summary>Click to reveal answer</summary>

```python
print("Name:", "Connor", "Age:", 35)
```

**Key points:**
- Separate items with commas
- Automatic space between items
- Can mix text and numbers
</details>

---

## Card 5: Special Characters

**Q: How do you print text on separate lines using \n?**

<details>
<summary>Click to reveal answer</summary>

```python
print("Line 1\nLine 2\nLine 3")
```

**Output:**
```
Line 1
Line 2
Line 3
```

**`\n` = new line character**
</details>

## Card 6: Fix the Bug

**Q: What's wrong with this code?**

```python
print(Hello World)
```

<details>
<summary>Click to reveal answer</summary>

**Missing quotes!**

```python
print("Hello World")
```

**Text must be in quotes**
</details>
