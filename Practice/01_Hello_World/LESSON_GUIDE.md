## üîÑ Step 3: Learn More (As Needed)

**Come back here ONLY if you're stuck and tried for 10+ min**

### More Print Features:

**New lines:**
```python
print("Line 1\nLine 2")
```

**Tabs:**
```python
print("Name:\tConnor")
```

**Multiple prints:**
```python
print("First")
print("Second")  # Automatic new line
```

**Common Errors:**

```python
# WRONG - Missing quotes
print(Hello)

# RIGHT
print("Hello")
```

```python
# WRONG - Mismatched quotes
print("Hello')

# RIGHT
print("Hello")
```
