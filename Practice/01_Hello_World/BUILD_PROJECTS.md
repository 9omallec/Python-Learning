## Project 1: Personal Introduction Card

**Goal:** Print a card about yourself

**Requirements:**
- Your name
- Your age
- Your city
- Your favorite hobby
- Use at least 5 print statements

**Example Output:**
```
====================
   ABOUT ME
====================
Name: Connor
Age: 35
City: New York
Hobby: Coding
====================
```

**Challenge:** Make it look nice with decorations (===, ---, etc.)

---

## Project 2: Daily Schedule

**Goal:** Print your daily schedule

**Requirements:**
- At least 5 time slots
- Activities for each time
- Make it readable

**Example Output:**
```
MY DAILY SCHEDULE
-----------------
8:00 AM  - Wake up
9:00 AM  - Breakfast
10:00 AM - Study Python
12:00 PM - Lunch
1:00 PM  - Build projects
```

**Challenge:** Add emoji or decorations

---

## Project 3: Simple Story

**Goal:** Print a short 5-line story

**Requirements:**
- Beginning, middle, end
- Use \n for some line breaks
- Make it interesting!

**Example:**
```
Once upon a time, there was a programmer.
They decided to learn Python.

After 30 days of practice...
They built amazing projects!
```

**Challenge:** Make it funny or personal

---

## Project 4: ASCII Art Banner

**Goal:** Create text art

**Requirements:**
- Your name in large letters
- Or a simple picture (smiley face, house, etc.)
- Use print statements creatively

**Example:**
```
  _____
 |     |
 |  o  |
 | --- |
 |_____|
  HELLO
```

**Challenge:** Make it as creative as possible

---

## Project 5: Recipe Card

**Goal:** Print a simple recipe

**Requirements:**
- Recipe name
- Ingredients (at least 4)
- Steps (at least 3)
- Format nicely

**Example Output:**
```
======== RECIPE ========
Peanut Butter Sandwich

INGREDIENTS:
- 2 slices of bread
- Peanut butter
- Jelly (optional)

STEPS:
1. Spread peanut butter on one slice
2. Add jelly if desired
3. Put slices together
4. Enjoy!
========================
```
