## Project 8: Range Explorer (15 min)

**Goal:** Help understand how range() works

**Requirements:**
- Let user experiment with range
- Ask for start, stop, step
- Show what range produces
- Handle errors gracefully

**Example Output:**
```
Range Explorer!

Start: 1
Stop: 10
Step: 2

range(1, 10, 2) produces:
1, 3, 5, 7, 9

Try again? (y/n): y

Start: 10
Stop: 0
Step: -2

range(10, 0, -2) produces:
10, 8, 6, 4, 2
```
