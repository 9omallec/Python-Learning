## ğŸ”„ Step 3: Learn More (As Needed)

**Come back ONLY if stuck after trying 10+ min**

### Data Types:

```python
age = 35              # int (whole number)
price = 19.99         # float (decimal)
name = "Connor"       # str (string/text)
is_student = True     # bool (True/False)
```

### Math with Variables:

```python
x = 10
y = 3

print(x + y)    # 13 (addition)
print(x - y)    # 7 (subtraction)
print(x * y)    # 30 (multiplication)
print(x / y)    # 3.333... (division)
print(x // y)   # 3 (integer division)
print(x % y)    # 1 (remainder/modulo)
print(x ** y)   # 1000 (exponent/power)
```

### F-Strings (Best way to print variables):

```python
name = "Connor"
age = 35
print(f"Hi, I'm {name} and I'm {age}")
```

### Common Errors:

```python
# WRONG - Missing quotes for text
name = Connor

# RIGHT
name = "Connor"
```

```python
# WRONG - Can't start with number
2nd_place = "Silver"

# RIGHT
second_place = "Silver"
```
